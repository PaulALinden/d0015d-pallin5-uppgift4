# Flödesschema för webbplatsens informationsstruktur

Detta flödesschema visar informationsstrukturen för webbplatsen, inte länkstrukturen.

## Webbplats: Styrka Gym

```text
Styrka Gym
|
+-- Global meny
|   |
|   +-- Startsida
|   |   +-- Presentation av gymmet
|   |   +-- Öppettider
|   |   +-- Snabb överblick
|   |
|   +-- Träning & utbud
|   |   +-- Gym
|   |   +-- Personlig träning
|   |
|   +-- Gruppträning
|   |   +-- Schema
|   |
|   +-- Priser
|   |
|   +-- Om gymmet
|       +-- Lokaler
|       +-- Personal
|
+-- Hjälp-meny
|   +-- Kontakt
|   +-- Öppettider
|
+-- Sidfot
    +-- Snabblänkar
    +-- E-post
    +-- Telefon
    +-- Adress
```

## Sammanfattning

**Global meny:** Startsida, Träning & utbud, Gruppträning, Priser, Om gymmet

**Undernivåer i innehållet:** Träning & utbud -> Gym, Personlig träning. Gruppträning -> Schema. Om gymmet -> Lokaler, Personal.

**Hjälp-meny:** Kontakt, Öppettider

**Sidfot:** Snabblänkar, e-post, telefon, adress
